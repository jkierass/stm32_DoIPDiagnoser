Attention

This is the main file of the project "DoIP Diagnoser". It consists of two programs running on CM4 and CM7 core separately. This core, i.e. CM4, is reponsible for handling ethernet port communication with TCP/IP stack using LwIP library to communicate with a car's ECU in order to fetch diagnostic data through DoIP protocol. It also handles measuring current air temperature using MCP9808 I2C Grove sensor.

Whole projects uses FreeRTOS as an operating system. IPC implementation is based on FreeRTOS message buffers (xMessageBuffer).

Most of the user implementation is written in C++, with elements in C.
