This is the main file of the project \char`\"{}\+Do\+I\+P Diagnoser\char`\"{}. It consists of two programs running on C\+M4 and C\+M7 core separately. This core, i.\+e. C\+M7, is reponsible for L\+CD screen handling (with Touch\+G\+FX framework used), G\+UI, as well as running R\+TC and \mbox{\hyperlink{class_calculator}{Calculator}} applications.

Whole projects uses Free\+R\+T\+OS as an operating system. I\+PC implementation is based on Free\+R\+T\+OS message buffers (x\+Message\+Buffer).

Most of the user implementation is written in C++, with elements in C.
