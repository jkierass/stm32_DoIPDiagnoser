Attention

This is the main file of the project "DoIP Diagnoser". It consists of two programs running on CM4 and CM7 core separately. This core, i.e. CM7, is reponsible for LCD screen handling (with TouchGFX framework used), GUI, as well as running RTC and Calculator applications.

Whole projects uses FreeRTOS as an operating system. IPC implementation is based on FreeRTOS message buffers (xMessageBuffer).

Most of the user implementation is written in C++, with elements in C.
